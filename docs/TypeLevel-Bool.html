ifThenElse :: The Bool c -> ((c :~: True) -> a) -> ((c :~: False) -> a) -> a Source #

For use with -XRebindableSyntax. This function can be used to make Haskell look reasonably dependent:

depHask :: The Bool x -> IfThenElse x Int String
depHask cond =
    if cond
        then \Refl -> 1
        else \Refl -> "abc"
