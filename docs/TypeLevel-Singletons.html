TypeLevel.Singletons

Description

Provides singletons and general type-level utilities. singletons are value-level representations of types.

Eisenberg, Richard A., and Stephanie Weirich. “Dependently Typed Programming with Singletons.” In Proceedings of the 2012 Haskell Symposium, 117–130. Haskell ’12. New York, NY, USA: ACM, 2012. doi:10.1145/2364506.2364522.

    This is just a newtype wrapper for Natural. As such, it is only valid if the programmer can't construct values where the type index doesn't match the contained value. For that reason, the constructor is not exported.

    The reason for this setup is to allow properties and invariants to be proven about the numbers involved, while actual computation can be carried out efficiently on the values at runtime.
